# TheFinals Logitech Macros / TheFinals 罗技宏文件

## English Version

### Project Description
This project contains Logitech macro configurations for TheFinals game. Please note that some scripts may be outdated due to game updates. Users should test the functionality themselves before use.

### Features
- Custom macro configurations for Logitech devices
- Optimized for TheFinals gameplay mechanics
- Multiple weapon support

### Prerequisites
- Logitech G Hub software installed
- Compatible Logitech gaming device (mouse/keyboard)

### Installation
1. Download the macro files from this repository
2. Open Logitech G Hub
3. Import the macro files into your profile
4. Assign macros to your preferred keys/buttons

### Usage Notes
⚠️ **Important**: Some scripts may no longer work due to game updates.
